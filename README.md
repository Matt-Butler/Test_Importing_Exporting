# Test_Importing_Exporting

# Install
```
npm install
```

# Build
```
npm run compile
```

# Example 1
```
npm run example_1
```

Example of importing using this syntax:
```
import { print_a } from './print';
```
Notice how it imports print_a, print_b, and print_c

# Example 2
```
npm run example_2
```

Example of importing using this syntax:
```
import print_a from './print_a';
```
Notice how it only imports print_a
